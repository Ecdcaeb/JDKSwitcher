# JDKSwitcher

 A software which enables you change the "JAVA_HOME" fastly.

Use `jdks.cfg` to add custom jdk locations.

for example:
```cfg
D:\Program Files\JetBrains\IntelliJ IDEA Community Edition 2022.3.3\jbr
D:\Program Files\JetBrains\IntelliJ IDEA Community Edition 2022.3.3\jbr
D:\Program Files\JetBrains\IntelliJ IDEA Community Edition 2022.3.3\jbr
```
